Política de Privacidade do Aplicativo Backup SQL/MySQL

Última atualização: 19 de novembro de 2025

1. Coleta e Uso de Dados

Este aplicativo é uma ferramenta de backup de desktop e não possui servidores próprios. Todos os dados coletados são usados exclusivamente para a função de backup.

O aplicativo não coleta, armazena em servidores de terceiros ou compartilha dados pessoais dos usuários.

2. Acesso à API do Google Drive (Escopo Restrito)

Para fornecer o serviço de backup em nuvem, o aplicativo solicita acesso à API do Google Drive. O acesso é estritamente limitado:

Escopo Utilizado: https://www.googleapis.com/auth/drive.file

Uso dos Dados: Este escopo permite que o aplicativo apenas crie, edite e exclua os **arquivos de backup que ele mesmo criou**. O aplicativo não tem acesso para ler ou modificar outros arquivos, pastas ou dados confidenciais do seu Google Drive. O acesso é utilizado exclusivamente para subir e manter a integridade dos arquivos compactados de backup.

Os dados do seu Google Drive nunca são acessados para fins de marketing, publicidade ou análise por terceiros.
